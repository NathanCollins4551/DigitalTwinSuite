Digital Twin Controls

WASD
Movement

Use WASD or arrow keys to navigate the floor.

RMB
Look Around

Hold Right Mouse Button and move mouse to rotate view.

LMB
Select

Left Click on machines to view real-time data and details.

T
Voice Chat

Hold T to speak with the Convai AI Assistant.

↵
Text Chat

Press Enter to type messages to the Convai AI.

Viewing Machine Data

To view the machine info dashboard, simply Left Click on any machine in the 3D view. This will instantly pull up the dashboard containing the live data stream for that specific unit.
